AlloyFL Tutorial
AlloyFL Execution

How to execute AlloyFL?

Select the "AlloyFL" button to start an AlloyFL execution. The AlloyFL drop down menu allows you to customize (1) the weight between coverage-based and mutation-based fault localization and (2) the suspiciousness score.
The weight ranges from 0 to 1. Users can select in increments of 0.1.
For suspciousness formulas, the user is presented with a selectable list with our 5 supported formulas.

Output

The AlloyFL tab reports a list of suspicious locations form most to least suspicious. To start, all suspicious locations are highlighted in the original model. Deep red means a location is very suspicious while yellow means a location is only slightly suspicious. For the ranked list, for each location, you can view: (1) the formula, (2) the score, and (3) a highlight of the AST node in the original model. Viewing an individual AST node highlight removes all other highlighting from the model.

If the user desires, there is a text prompt on the main page to allow the user to re-highlight the entire model.
