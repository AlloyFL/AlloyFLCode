AlloyFL Tutorial

This tutorial will outline key concepts of AlloyFL, a fault localization technique for Alloy.

This user guide is divided into the following sections:
     0. Motivation: Outlines the design choices of and motivation for AlloyFL.
     1. Setting up a Fault Localization Problem: Outlines what conditions are needed for AlloyFL to run.
     2. AlloyFL Execution: Outlines how to use AlloyFL and the menu options.
